Mysql

  • Insert: insert into table (column1, column2) values (value1, value2);
  • 获取当前时间函数：now()
  • 模糊匹配：like ‘%xxx%’
  • DDL:
    • 增加列：alter table table_name add column;
    • 删除列：alter table table_name drop column;
    • 修改列：alter table table_name modify column column_name [attribute];
    • 删除表：drop table table_name;
  • 查看建表语句：show create table table_name;
  • source: 数据库导入命令
    • 确保脚本文件的字符编码是utf8
    • 在文件第一行前加入：set names utf8
  • 加密函数：
    • PASSWORD(): 创建一个经过加密的密码字符串，加密过程不可逆
    • ENCRYPT(, ): 使用UNIX crypt()加密字符串
    • ENCODE(, ) DECODE(, ): 加密解密字符串，两个参数为被加密或解密的字符串以及密钥
    • MD5(): 计算字符串的MD5校验和（128位）
    • SHA5(): 计算字符串的SHA5校验和（160位）
