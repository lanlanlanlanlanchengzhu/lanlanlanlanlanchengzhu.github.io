MD5 概念和用途

  • 概念：Message-Digest-Algorithm 5（信息-摘要算法）

    任何文件都有且只有一个唯一的MD5信息值。文件被修改后，其MD5信息值也会改变

  • 用途：用于校验文件是否被修改过或两个文件是否一致

Base64 概念和作用

  • 将8位的非英语字符转换为7位的ASCII字符

  • Base64，就是选出64个字符——小写字母a-z、大写字母A-Z、数字0-9、符号”+”、”/“（再加上作为垫底的“=”，实际上是65个字符）作为一个基本字符集。然后，所有其他符号都用此字符集进行编码

  • 转换步骤

    1. 将每三个字节作为一组，一共是24个二进制位

    2. 将这24个二进制位分为四组，每个组有6个二进制位

    3. 在每组前面加00，扩展成32个二进制位，即四个字节

    4. 根据下表，得到扩展后每个字节的对应符号，这就是Base64的编码值

  • Base64将三个字节转换为四个字节，因此Base64编码后的文本会比原文本大三分之一左右
